This contribution is a Compare Products feature that adds a checkbox under the products image for the listing page. When checking the box it will add it to the session as something the customer will want to compare later with other products.

Company: Do Creative Design
Author: Brit
Version: 1.0.0
Last Updated: May 18, 2011
Zen Cart Version: v1.3.9
PHP Compatible: > 5

I have setup a demo cart with it plugged in. The cart is is a default install of Zen cart 1.3.9 with the demo pack.

http://www.docreativedesign.com/shop/zen-cart_01/

To install change the folder YOUR_TEMPLATE to the appropriate name. Then just upload the files to your store.

The run.sql.txt file can be ran via the admin in admin->tools->install sql patches

This contribution is a Compare Products feature that adds a button under the products image for the listing pages. When clicking the button it will add it to the session as something the customer will want to compare later with other products.

Company: Zen4All
Author: Erik
Version: 2.0.0
Last Updated: November 10, 2018
Zen Cart Version: v1.5.5f
PHP Compatible: ≥ 5.6

I have setup a demo cart with it plugged in. The cart is a default install of Zen cart 1.5.5f with the demo pack.

Zen4All Zen Cart Demo Shop

To install change the folder YOUR_TEMPLATE to the appropriate name. Then just upload the files to your store.

The run.sql.txt file can be ran via the admin in admin->tools->install sql patches

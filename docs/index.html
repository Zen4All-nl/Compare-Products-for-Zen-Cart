What does this mod do?

This contribution is a Compare Products feature that adds a compare button to the products listing pages. When clicking the button it will add it to the session as something the customer will want to compare later with other products.

Compare buttons can be shown on multiple pages. Currently these pages are:

  • Product info detail pages (all default product types)
  • Featured listing
  • All products listing
  • New products listing

New install

BACKUP FILES AND DATABASE

Copy the files into the correct directories.
The files marked with the YOUR_TEMPLATE_NAME path need to go into your template directories, NOT into directories literally called "YOUR_TEMPLATE_NAME".

There are some database changes these will auto install as soon as you log in to your admin.

Upgrade

When upgrading from a previous version, please use the file list as a guide to see which file have been changed. After applying the file changes, log in to your Zen Cart admin, to complete the installation. The database changes, if needed, will be executed automatically

Un-install

Use the file list to remove all the files, and changes made for this mod. Then log in to your Zen Cart Admin, and run the product_finder_uninstall.sql (found in the uninstall folder) in Admin=>Tools=>Install SQL Patches

Configuration

After copying all the file to your server you need to open your shops admin panel. This will install the needed database changes.
Next you need to enable the compare module. To do this you need to go to "Configuration => Compare Products".

The first value to enable is the main switch: Enable Product Compare. This will turn the whole module on or off.

The next values to set are the switches for the different pages. These will turn the display on the different Product Listing Pages on or off.

There also a setting to sethow many prodcts can be compared at the same time.

A final setting is used to set howmany characters of the products description should be displayed on the compare page.
